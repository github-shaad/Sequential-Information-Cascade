# Sequential Information Cascade: SPNE Solver

A Python solver that calculates the Subgame Perfect Nash Equilibrium for a sequential guessing game. The game and the math is described in the pdf. 

## Requirements
* Python 3.x
* NumPy
* Matplotlib
* Seaborn

## Usage

Install the required libraries and run the script:

```bash
pip install numpy matplotlib seaborn
python mystery_guest_solver.py
